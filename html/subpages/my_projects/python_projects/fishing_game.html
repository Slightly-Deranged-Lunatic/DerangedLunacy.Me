What is this project?

This is a CLI fishing game. I originally started this when I was still relatively a beginner in Python, and it taught me a lot. However, as time went on I learned more and things started to become like spaghetti so I had to rewrite it pretty significantly. The rewrite made the code a lot more modular and clean because I was more experienced and knew what I was doing. This project continues to teach me a lot too and put stuff I learned into pratice. To this day it is one of the projects I am most proud of.
