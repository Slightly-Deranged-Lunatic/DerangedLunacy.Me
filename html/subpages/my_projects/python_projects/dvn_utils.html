What is this project?

Honestly this project really wasn't much. I had a lot of these scripts separated across repositories already, and I just felt like having one central repository for all of this would be better. DVN stood for Dummies Vs Noobs (a game on Roblox) that I don't play much anymore for reasons I won't get into, and these are just a list of scripts that would do something in relation to the game. These scripts weren't hard to do or anything, I just wanted something to spend my time on so I made them.
